# Getting started with thread

Thanks for using thread! I hope you find it useful for your projects.
Here's to you get started.


<Steps>

### Prerequisites

* Python 3.9+



### Installing

#### Published Release

We recommend using a packaging manager like `pip` or `poetry`.

<Tabs items={['pip', 'pipx', 'poetry']}>
  <Tabs.Tab>
    ```sh
    pip install thread
    ```
  </Tabs.Tab>

  <Tabs.Tab>
    ```sh
    pipx install thread
    ```
  </Tabs.Tab>

  <Tabs.Tab>
    ```sh
    poetry add thread
    # OR
    poetry install thread
    ```
  </Tabs.Tab>
</Tabs>


#### Building From Source
```sh
# Clone this repository
git clone https://github.com/python-thread/thread

# Install the upstream package
pip install -e .
```

#### Development

```sh
# Clone this repository
git clone https://github.com/python-thread/thread

# Install poetry
pip install poetry

# Install dependencies
poetry install
```



### Importing thread

Import thread into your .py file
```py
import thread
```
